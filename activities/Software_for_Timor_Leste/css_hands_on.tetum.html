Bem-vindo ao Estilo CSS

Ne'e parágrafu normal. Tenta ho propriedade CSS sira iha tag estilo atu haree sira nia impaktu ba aspeto tekstual ida-ne'e.

Parágrafu ida-ne'e aplika klase "destake". Tenta muda regra .destake iha CSS atu hamosu mudansa ba nia estilo.

Parágrafu ida-ne'e iha id úniku "special-paragraph". Ita bele alvo ida-ne'e espesifikamente ho CSS liuhosi #special-paragraph.
