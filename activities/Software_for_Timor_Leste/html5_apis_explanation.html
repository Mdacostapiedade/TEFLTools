Step 1: Implement Geolocation API

The Geolocation API allows retrieving the user's current location. Here's how the code works:

function getLocation() {
    if (navigator.geolocation) {
        navigator.geolocation.getCurrentPosition(showPosition);
    } else {
        document.getElementById("location").innerHTML = "Geolocation is not supported by this browser.";
    }
}

function showPosition(position) {
    document.getElementById("location").innerHTML = "Latitude: " + position.coords.latitude +
    "<br>Longitude: " + position.coords.longitude;
}

Explanation:

  • The getLocation() function checks if the browser supports geolocation using navigator.geolocation.
  • If supported, it calls getCurrentPosition() and passes the showPosition function as a callback.
  • The showPosition() function receives the position object and displays the latitude and longitude.

Step 2: Implement Local Storage API

The Local Storage API allows saving and retrieving data locally in the browser. Here's how the code works:

function saveName() {
    var name = document.getElementById("name").value;
    localStorage.setItem("name", name);
    document.getElementById("savedName").innerHTML = "Saved name: " + name;
}

window.onload = function() {
    var name = localStorage.getItem("name");
    if (name) {
        document.getElementById("savedName").innerHTML = "Saved name: " + name;
    }
};

Explanation:

  • The saveName() function retrieves the value of the input field with the id "name".
  • It saves the name using localStorage.setItem(), with "name" as the key and the input value as the value.
  • The saved name is then displayed using document.getElementById("savedName").innerHTML.
  • On page load, the window.onload event retrieves the saved name using localStorage.getItem("name").
  • If a name exists, it is displayed using document.getElementById("savedName").innerHTML.

Step 3: Implement Drag and Drop API

The Drag and Drop API enables dragging and dropping elements. Here's how the code works:

function allowDrop(ev) {
    ev.preventDefault();
}

function drag(ev) {
    ev.dataTransfer.setData("text", ev.target.id);
}

function drop(ev) {
    ev.preventDefault();
    var data = ev.dataTransfer.getData("text");
    ev.target.appendChild(document.getElementById(data));
}

Explanation:

  • The allowDrop() function is called when an element is dragged over a drop target. It prevents the default behavior.
  • The drag() function is called when an element starts being dragged. It sets the data being dragged using ev.dataTransfer.setData().
  • The drop() function is called when a dragged element is dropped onto a drop target.
  • It retrieves the dropped data using ev.dataTransfer.getData() and appends the dragged element to the drop target using ev.target.appendChild().

Step 5: Enhance Microdata markup

Microdata allows adding semantic markup to your HTML. Here's an example of enhanced Microdata markup for a product:

<div itemscope itemtype="http://schema.org/Product">
    <h3 itemprop="name">Product Name</h3>
    <img itemprop="image" src="product.jpg" alt="Product Image">
    <p itemprop="description">Product description goes here.</p>
    <p>Price: <span itemprop="price">$19.99</span></p>
    <p>In stock: <span itemprop="availability" href="http://schema.org/InStock">Yes</span></p>
    <p>Rating: <span itemprop="ratingValue">4.5</span> out of <span itemprop="bestRating">5</span></p>
</div>

Explanation:

  • The itemscope attribute indicates that the <div> element contains Microdata.
  • The itemtype attribute specifies the schema URL for the item, in this case, "http://schema.org/Product".
  • The itemprop attributes are used to identify the specific properties of the product, such as name, image, description, price, availability, and rating.

    margin: 5px 0;

Explanation:

  • The .product class styles the container element with a border, padding, and margin.
  • The .product img selector limits the maximum width of the product image and maintains its aspect ratio.
  • The .product p selector adds margin to the paragraphs inside the product container.
