Esplikasaun Kódigu API HTML5 no Mikrodadus

Pasu 1: Implementa API Geolokalizasaun

API Geolokalizasaun permite recupera lokalizasaun atual uzuariu. Ne'e oinsa kódigu servisu:

function getLocation() {
    if (navigator.geolocation) {
        navigator.geolocation.getCurrentPosition(showPosition);
    } else {
        document.getElementById("fatin").innerHTML = "Geolokalizasaun la suporta husi ne'e browser.";
    }
}

function showPosition(posisaun) {
    document.getElementById("fatin").innerHTML = "Latitude: " + posisaun.coords.latitude +
    "<br>Longitude: " + posisaun.coords.longitude;
}

Esplikasaun:

  • Funsaun getLocation() verifica se browser suporta geolokalizasaun liu husi navigator.geolocation.
  • Se suporta, nia halo kasa getCurrentPosition() no pasa funsaun showPosition hanesan callback.
  • Funsaun showPosition() simu objetu posisaun no hatudu latitude no longitude.

Pasu 2: Implementa API Armazenamentu Lokal

API Armazenamentu Lokal permite salva no recupera dadus lokálmente iha browser. Ne'e oinsa kódigu servisu:

function saveName() {
    var naran = document.getElementById("narán").value;
    localStorage.setItem("narán", naran);
    document.getElementById("naránSalvu").innerHTML = "Naran salvu: " + naran;
}

window.onload = function() {
    var naran = localStorage.getItem("narán");
    if (naran) {
        document.getElementById("naránSalvu").innerHTML = "Naran salvu: " + naran;
    }
};

Esplikasaun:

  • Funsaun saveName() recupera valor husi kampu input ho id "narán".
  • Nia salva naran liu husi localStorage.setItem(), ho "narán" hanesan xave no valor input hanesan valor.
  • Naran ne'ebé salvu depois hatudu liu husi document.getElementById("naránSalvu").innerHTML.
  • Iha momentu karga pájina, eventu window.onload recupera naran salvu liu husi localStorage.getItem("narán").
  • Se naran eziste, nia hatudu liu husi document.getElementById("naránSalvu").innerHTML.

Pasu 3: Implementa API Rasta no Hatuun

API Rasta no Hatuun permite rasta no hatuun elementus. Ne'e oinsa kódigu servisu:

function allowDrop(ev) {
    ev.preventDefault();
}

function drag(ev) {
    ev.dataTransfer.setData("testu", ev.target.id);
}

function drop(ev) {
    ev.preventDefault();
    var dadus = ev.dataTransfer.getData("testu");
    ev.target.appendChild(document.getElementById(dadus));
}

Esplikasaun:

  • Funsaun allowDrop() chama bainhira elementu rastadu liu husi alvu hatuun. Nia prevene behavior default.
  • Funsaun drag() chama bainhira elementu hahu rasta. Nia define dadus
