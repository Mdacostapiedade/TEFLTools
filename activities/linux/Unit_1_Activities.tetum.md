'# Atividades Envolventes ba Introdusaun ba Linux no Linha Komandu

## Atividades Aquecimento

### 1. Jogo Adivinha Logo Linux
- Apresenta logos distribuisaun Linux oioin iha projetor
- Alunu sira servisu iha par hodi hatene naran distribuisaun sira-nian
- Diskute distribuisaun ne'ebé populár liu no sira nia karakterístika

### 2. "Hau Haklaken" Edisaun Linha Komandu
- Instrutor buka terminal ida no navega liu liu husi pasta sira
- Alunu sira hatoo komandu sira-ne'ebé sira hatene ka bele adivinha husi saida mak hatudu

## Atividades Prinsipal Aula

### 1. Buska Tesouru iha Linha Komandu
- Fornese alunu sira ho lista item hodi buka uza ferramenta linha komandu
- Ezemplu: file boot liu iha pasta uma, numeru utilizador iha sistema, versaun kernel atuál
- Alunu sira kompiti hodi buka item hotu-hotu uza de'it komandu CLI

### 2. Ofisina Harii Komandu
- Apresenta rezultadu ne'ebé hakarak (ex., "lista file .txt hotu iha pasta atuál")
- Alunu sira kolabora hodi konstrui komandu loos hodi hetan rezultadu

## Tarefa Trabalha iha Grupu ka Par

### 1. Arkitektu Estrutura Pasta
- Iha grupu 3-4, alunu sira desenha estrutura pasta ba kompañia simula
- Grupu sira troka desenho no implementa sira uza de'it komandu CLI

### 2. Pictionary Linha Komandu
- Alunu ida husi kada par simu komandu ida hodi dezenha
- Parseiru tenke adivinha komandu husi dezenha
- Troka papel no repete ho komandu foun

## Exersisiu Prátika Individual

### 1. Labirintu Linha Komandu
- Fornese alunu sira ho file textu ida ne'ebé inklui labirintu pasta sira
- Alunu sira tenke navega labirintu uza komandu cd no ls hodi buka file ida ne'ebé subar

### 2. Komandu Fogo Rápidu
- Hatudu lista tarefa iha projetor (ex., "kria pasta 'projetus'")
- Alunu sira kompiti hodi kompleta tarefa hotu uza komandu loos

## Atividades Refreska ka Remata

### 1. Jeopardy Linha Komandu
- Kria jogo Jeopardy ho kategoria sira hanesan "Operasaun File," "Navigasaun," no "Info Sistema"
- Alunu sira kompiti iha grupu hodi responde pergunta sira ho komandu loos

### 2. Fortune Cookies Linux
- Uza komandu 'fortune' hodi produz frase ka ditadu aleatóriu
- Alunu sira le'e sira nia "fortune" no esplika oinsá relasiona ho buat ne'ebé sira aprende kona-ba Linux

### 3. Papel Minutu Ida
- Alunu sira hakerek durante minutu ida kona-ba:
  1. Asuntu importante liu-ne'ebé sira aprende
  2. Pergunta ida ne'ebé sira sei iha kona-ba Linux ka linha komandu
