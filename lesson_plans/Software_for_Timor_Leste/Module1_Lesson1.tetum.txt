'# Plano Aula: Introdusaun ba Ficheirus Textu no Báziku HTML

## Durasaun: 1 semana (5 sesaun, 2 oras kada ida)

### Sesaun 1: Introdusaun ba Ficheirus Textu

Rekursus:
Haree ficheirus iha Leitura no Skriptus relasionadu ho HTML.
M1L1_TL_fun_facts.html iha ficheiru HTML báziku ho elementus iha lisaun ida ne'e.

#### Objetivus:
- Komprende saida mak ficheirus textu no importansia sira iha programasaun
- Aprende kria no edita ficheirus textu uza editór textu simples

#### Materiais:
- Komputadór ho editór textu báziku (exemplu, Notepad, TextEdit)
- Folhetus kona-ba formatu ficheirus textu no kodifikasaun

#### Aktividades:
1. (15 min) Introdusaun ba ficheirus textu: definisaun, utilizasaun, no formatu
2. (30 min) Demonstrasaun: Kria no edita ficheirus textu
3. (45 min) Pratika: Estudantes kria no edita sira nia ficheirus textu
4. (20 min) Diskusaun: Formatu diferente ficheirus textu no utilizasaun sira
5. (10 min) Perguntas & Respostas no enseramentu

#### Tarefa de Casa:
Kria biografia simples iha ficheiru textu, inklui naran, idade, no intereses

### Sesaun 2: Introdusaun ba HTML

#### Objetivus:
- Komprende saida mak HTML no papel ida nian iha dezenvolvimentu web
- Aprende estrutura HTML báziku no tags komuns

#### Materiais:
- Komputadór ho editór textu no navegador web
- Folhetus "HTML cheat sheet"

#### Aktividades:
1. (20 min) Introdusaun ba HTML: Saida mak ida ne'e, oinsa ida ne'e funsiona
2. (30 min) Demonstrasaun: Estrutura HTML báziku no tags komuns
3. (50 min) Pratika: Estudantes kria pájina HTML simples
4. (15 min) Haree HTML iha navegador web
5. (5 min) Perguntas & Respostas no enseramentu

#### Tarefa de Casa:
Konverte biografia pessoál husi tarefa anterior ba formatu HTML

### Sesaun 3: Elementus HTML no Atributus

#### Objetivus:
- Aprende kona-ba elementus HTML diferente no objetivu sira
- Komprende no uza atributus HTML

#### Materiais:
- Komputadór ho editór textu no navegador web
- Folhetus kona-ba elementus HTML komuns no atributus

#### Aktividades:
1. (20 min) Revizaun HTML báziku no introduzaun ba elementus foun
2. (25 min) Demonstrasaun: Uza elementus HTML no atributus
3. (60 min) Pratika: Estudantes kria pájina HTML ho elementus barak
4. (10 min) Revizaun par: Estudantes revizaun pájina HTML malu sira nian
5. (5 min) Perguntas & Respostas no enseramentu

#### Tarefa de Casa:
Kria pájina HTML kona-ba Timor-Leste, inklui titulu, paragrafu, no lista

### Sesaun 4: Ligasaun HTML no Imajen

#### Objetivus:
- Aprende kria hyperlink iha HTML
- Komprende oinsa enkorpora imajen iha pájina HTML

#### Materiais:
- Komputadór ho editór textu no navegador web
- Imajen amostra ba enkorporasaun

#### Aktividades:
1. (15 min) Introdusaun ba hyperlink no importansia sira
2. (20 min) Demonstrasaun: Kria ligasaun interna no externa
3. (25 min) Pratika: Estudantes aumenta ligasaun ba pájina Timor-Leste sira nian
4. (20 min) Introdusaun ba enkorpora
