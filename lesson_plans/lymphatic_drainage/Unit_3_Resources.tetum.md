Anatomia Sistema Linfátiku Isin Leten

Kakorok no Kabaas
- Grupu node linfátiku boot:
  * Node servikal (anterior no posterior)
  * Node supraclavicular
  * Node axilar
- Dalan drenajen:
  * Husi oin no ulun ba node servikal
  * Husi kakorok ba node supraclavicular
  * Husi kabaas ba node axilar

Liman no Liman-fuan
- Grupu node linfátiku boot:
  * Node epitrochlear (laran kotuk liman)
  * Node axilar
- Dalan drenajen:
  * Husi liman-fuan no antebraço ba node epitrochlear
  * Husi liman leten no node epitrochlear ba node axilar

Sekuénsia MLD ba Kakorok no Kabaas

1. Toke preparatóriu:
   - Movimentu mamar, husi sternum ba node axilar
   - Movimentu sirkulár suave liu husi node supraclavicular

2. Drenajen Kakorok:
   - Movimentu mamar, husi linhajen maxilar ba node servikal
   - Toke mamar husi node servikal ba node supraclavicular

3. Drenajen Kabaas:
   - Movimentu mamar, husi lâmina kabaas ba node axilar
   - Toke sirkulár iha junta kabaas, move ba node axilar

Sekuénsia MLD ba Liman no Liman-fuan

1. Toke preparatóriu:
   - Movimentu mamar husi munheka ba node axilar
   - Movimentu sirkulár suave liu husi node axilar

2. Drenajen Liman Leten:
   - Movimentu mamar, husi kotuk liman ba node axilar
   - Toke mamar husi aspetu laran no liur liman leten

3. Drenajen Antebraço:
   - Movimentu mamar, husi munheka ba kotuk liman
   - Toke sirkulár iha kotuk liman, move ba node epitrochlear

4. Drenajen Liman-fuan:
   - Toke mamar husi ponta liman-fuan ba munheka
   - Movimentu sirkulár suave iha dala liman-fuan no kotuk liman-fuan

Konsiderasaun Kulturál iha Timor-Leste

1. Preokupasaun modestia:
   - Uza teknika draping atu mantein privasidade
   - Esplika prosedimentu kompletamente antes la'o kobre kualker área

2. Sensibilidade jéneru:
   - Preferénsia ba prátikante jéneru hanesan bainhira posivel
   - Prezensa membru família ka chaperone se presiza

3. Integrasaun kurasaun tradisionál:
   - Rekonhese krensa lokál kona-ba bubu no movimentu líkidu
   - Inkorpora konseitu familiar (ex., "ajuda bee isin nian hodi sai")

4. Tabu toka:
   - Konsiensia kona-ba área konsidera inapropriadu atu toka
   - Esplika nesesidade médika enkuantu respeita norma kulturál

5. Estilu komunikasaun:
   - Uza forma respeitozu ba enderesa
   - Uza komunikasaun indireitu bain
